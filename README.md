## 功能介绍（完整见运行截图）
管理员：用户注册、登录和管理；课程安排与时间表更新；泳池预约管理；财务统计和收入支出分析；系统公告发布；会员卡信息管理和销售。  
会员：注册并登录系统；查看和编辑个人信息；浏览泳装及泳池课程信息；在线预约泳池及课程；接收系统公告和通知；查询与购买会员卡。
